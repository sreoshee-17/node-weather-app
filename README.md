
# Node-Weather-App

A basic weather app built using NodeJS and ExpressJS


## Tech Stack

**Client:** HTML/CSS, JS

**Server:**  NodeJS, ExpressJS, HBS


## Features

* Displays date, location, weather condition and temperature using OpenWeatherAPI
